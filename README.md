# 🏦 FinPulse AI Ecosystem: Financial Intelligence Dashboard

FinPulse is an enterprise-grade financial analytics dashboard designed to provide real-time insights into transaction data. Built with **Python** and **Streamlit**, it features a dynamic currency engine and risk monitoring tools.



## 🌟 Key Features
- **Dynamic Currency Switching:** Instantly toggle between **USD ($)** and **INR (₹)** across the entire dashboard.
- **Executive KPIs:** Real-time calculation of Total Volume, Average Transaction Value, and Success Rates.
- **Risk & Fraud Detection:** Interactive threshold filtering to identify high-value anomalies and potential risks.
- **Advanced Visualizations:** Interactive Pie and Bar charts for Channel and Transaction Type analysis.
- **Large Scale Data:** Optimized to handle and process over **10,000+ transaction records** seamlessly.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** Pandas (Data Processing), Plotly (Interactive Charts), Streamlit (Web Interface)
- **Deployment:** Streamlit Cloud

## 📂 Project Structure
- `app.py`: The main application logic and UI code.
- `data.csv`: The core dataset containing 10,000+ financial transactions.
- `requirements.txt`: List of dependencies required to run the app.
